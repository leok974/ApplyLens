# ApplyLens Web

> **Note:** This is part of the ApplyLens monorepo. For agent roles and safe change boundaries (UI vs API vs tests), see [AGENTS.md](../../AGENTS.md) and .github/agents/*.md at the repo root.

React + Vite frontend for ApplyLens job inbox application.

## Development

```bash
pnpm install
pnpm dev
```

The app runs at http://localhost:5173

## Production Build

```bash
pnpm build
```

Built files will be in dist/.
